== Description ==

Offers you the power to enable and disable Gzip compression on your WordPress site.

Enabling GZIP compression improves UX and SERP rankings by decreasing page loading times. You can see the results as better scores in Google PageSpeed Insights, GTmetrix, and other website analyzers.

Supports Apache HTTP Server.
